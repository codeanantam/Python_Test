# Variables aur Data Types

## Variables Kya Hain?

Socho variable ek dabba hai jisme hum kuch rakh sakte hain. Dabbe pe naam likha hota hai aur andar value hoti hai.

```python
movie = "Dangal"
rating = 8.5
```

Yahan `movie` ek dabba hai jisme `"Dangal"` rakha hai, aur `rating` mein `8.5`.

## Variable Banana

Python mein variable banana bahut simple hai — bas naam likho, `=` lagao, aur value do:

```python
college = "Code Anantam"
semester = 3
percentage = 82.5
passed = True
```

Print karne ke liye:

```python
print(college)
print(semester)
```

## Variable Ke Naam Kaise Rakhe?

Kuch rules hain:

- Letters, numbers aur `_` use kar sakte ho
- Number se start nahi ho sakta
- Python ke special words (jaise `if`, `for`) use nahi kar sakte
- `name` aur `Name` alag hain (case-sensitive)

```python
# Sahi ✅
student_name = "Raj"
roll_no = 45

# Galat ❌
# 2nd_name = "Test"    → number se start
# class = "BCA"        → keyword hai
```

**Tip:** Variable names hamesha meaningful rakho. `x = 50` se better hai `marks = 50`.

## Data Types Samajho

Python mein 4 basic data types hain:

### Integer — Poore Numbers

```python
seats = 60
floor = 3
negative_marks = -5
```

Ye numbers bina decimal ke hote hain.

### Float — Decimal Wale Numbers

```python
temperature = 37.2
wallet_balance = 1250.75
```

### String — Text Data

Quotes mein likha hua kuch bhi string hai:

```python
branch = "Computer Science"
nickname = 'Coder'
```

Single ya double quotes — dono chalte hain.

### Boolean — Haan ya Na

Sirf do values: `True` ya `False`

```python
is_present = True
has_backlog = False
```

## Type Check Karna

`type()` function se pata chal jaata hai ki variable ka type kya hai:

```python
fees = 45000
name = "Amit"
cgpa = 7.8
active = True

print(type(fees))     # <class 'int'>
print(type(name))     # <class 'str'>
print(type(cgpa))     # <class 'float'>
print(type(active))   # <class 'bool'>
```

## Value Badalna

Variable ki value kabhi bhi badal sakte ho:

```python
level = 1
print(level)   # 1

level = 5
print(level)   # 5
```

Purani value khatam, nayi value aa gayi.

## Ek Line Mein Multiple Variables

```python
x, y, z = 10, 20, 30
print(x)   # 10
print(y)   # 20
```

Sabko same value deni ho:

```python
a = b = c = 0
```

## Dynamic Typing

Python mein aapko type batane ki zaroorat nahi. Python khud samajh jaata hai:

```python
data = 100       # ye integer hai
data = "Hello"   # ab ye string hai
```

Ye feature Python ko flexible banata hai — ise **dynamic typing** kehte hain.

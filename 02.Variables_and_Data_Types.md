# Variables and Data Types

## Storing Information in Python

Every program needs to work with data — a user's name, their age, a product's price, or whether a task is completed. In Python, we use **variables** to hold these values in memory so we can access and manipulate them throughout our program.

Think of a variable like a **labeled box**. The label is the variable name, and the content inside is the value.

```python
product = "Laptop"
price = 72000
```

Here, `product` is a box labeled "product" containing the text `"Laptop"`, and `price` holds the number `72000`.

## Creating and Using Variables

In Python, you create a variable simply by assigning a value using `=`.

```python
college = "Code Anantam Institute"
semester = 4
cgpa = 8.75
is_active = True

print(college)
print(semester)
```

No need to write `int semester = 4` like in Java or C. Python figures out the type on its own — this is called **dynamic typing**.

## Naming Conventions

Choosing good variable names makes your code readable. Follow these rules:

| Rule | Valid | Invalid |
|------|-------|---------|
| Use letters, digits, underscore | `user_age`, `score1` | — |
| Cannot start with a digit | `item_2` | `2item` |
| No spaces allowed | `first_name` | `first name` |
| Cannot use reserved keywords | `total` | `for`, `class` |
| Case-sensitive | `Name ≠ name` | — |

**Best practice:** Use **snake_case** for variable names (words separated by underscores):

```python
monthly_salary = 45000
student_count = 120
```

## Data Types Explained

Python has several built-in data types. Let's understand the most essential ones.

### Integers (`int`)

Whole numbers — positive, negative, or zero. No decimal point.

```python
enrolled_students = 350
temperature = -5
floor_number = 0
```

Integers support very large values in Python without overflow:

```python
big_number = 99999999999999999
print(big_number)
```

### Floating-Point Numbers (`float`)

Numbers that have a decimal component.

```python
weight = 68.5
pi_value = 3.14159
discount_rate = 0.15
```

> **Note:** When you divide two integers, Python always returns a float:
> ```python
> result = 10 / 4  # result is 2.5
> ```

### Strings (`str`)

A sequence of characters enclosed in quotes. You can use single `'...'` or double `"..."` quotes.

```python
greeting = "Hello, welcome to the course"
language = 'Python'
```

**Multi-line strings** use triple quotes:

```python
address = """
42 MG Road
Ahmedabad, Gujarat
India
"""
```

### Booleans (`bool`)

Can only hold two values: `True` or `False`. These are crucial for decision-making in programs.

```python
has_submitted = True
is_expired = False
```

Booleans are often the result of comparisons:

```python
print(10 > 5)    # True
print(3 == 7)    # False
```

## Identifying Data Types with `type()`

When you're unsure about a variable's type, use the `type()` function:

```python
ticket_price = 250
event_name = "Tech Summit"
rating = 4.8
available = True

print(type(ticket_price))   # <class 'int'>
print(type(event_name))     # <class 'str'>
print(type(rating))         # <class 'float'>
print(type(available))      # <class 'bool'>
```

## Updating Variable Values

Variables can be reassigned at any point. The old value is simply replaced:

```python
level = 1
print(level)   # 1

level = 2
print(level)   # 2

level = level + 1
print(level)   # 3
```

You can also use **shorthand operators**:

```python
score = 100
score += 25    # same as score = score + 25
score -= 10    # same as score = score - 10

print(score)   # 115
```

## Assigning Multiple Variables

Python lets you assign several variables in a single line:

```python
x, y, z = 10, 20, 30
print(x, y, z)
```

Assign the same value to multiple variables:

```python
a = b = c = 0
```

### Swapping Values

In most languages, swapping requires a temporary variable. Python makes it effortless:

```python
left = "Red"
right = "Blue"

left, right = right, left

print(left)    # Blue
print(right)   # Red
```

# Variables and Data Types in Python

## 1. Introduction

When we write programs, we need to store data so we can use it later. In Python, we store data using **variables**.

Variables are the building blocks of any program.

## 2. What is a Variable?

A variable is simply a name that holds a value.

**Example:**

```python
name = "Sneha"
age = 19
```

Here:

- `name` holds the text `"Sneha"`
- `age` holds the number `19`

To display them:

```python
print(name)
print(age)
```

## 3. Rules for Variable Names

Follow these rules when naming variables:

- Can contain letters, numbers, and underscore `_`
- Cannot start with a number
- Cannot use Python keywords like `if`, `for`, `while`
- Python is case-sensitive (`name` and `Name` are different)

**Good examples:**

```python
first_name = "Amit"
marks1 = 85
total_price = 200
```

**Bad examples:**

```python
1name = "Raj"       # Cannot start with number
for = "loop"        # 'for' is a keyword
```

## 4. Dynamic Typing

In Python, you don't need to tell the type of a variable. Python figures it out on its own.

```python
x = 10        # Python knows this is integer
x = "Hello"   # Now Python treats it as string
```

This is called **dynamic typing**.

## 5. Data Types

### 5.1 Integer (`int`)

Whole numbers without decimal.

```python
age = 20
marks = 95
```

### 5.2 Float (`float`)

Numbers with decimal values.

```python
height = 5.8
price = 49.99
```

### 5.3 String (`str`)

Text written inside quotes.

```python
city = "Surat"
greeting = 'Hello'
```

Both single and double quotes work.

### 5.4 Boolean (`bool`)

Only two values: `True` or `False`

```python
is_student = True
has_passed = False
```

## 6. Checking Data Type

Use `type()` to check what type a variable is.

```python
name = "Python"
age = 25
height = 5.9
active = True

print(type(name))     # <class 'str'>
print(type(age))      # <class 'int'>
print(type(height))   # <class 'float'>
print(type(active))   # <class 'bool'>
```

## 7. Changing Variable Values

You can change a variable's value anytime.

```python
score = 40
score = 80
print(score)   # 80
```

The new value replaces the old one.

## 8. Multiple Variables in One Line

```python
a, b, c = 5, 10, 15
print(a, b, c)
```

Same value to multiple variables:

```python
x = y = z = 0
```

## 9. Common Mistakes

- Forgetting quotes around text:

```python
name = Rahul     # Error!
name = "Rahul"   # Correct
```

- Mixing string and number:

```python
age = "20"
print(age + 5)   # Error!
```

# Working with Strings

## Text is Everywhere

Almost every application deals with text — usernames, messages, emails, URLs, addresses. In Python, text is represented as **strings**. Mastering strings is essential for becoming a confident programmer.

## Creating Strings

Use single or double quotes — both are equivalent:

```python
company = "Code Anantam"
motto = 'Learn by Building'
```

For text that spans multiple lines, use triple quotes:

```python
bio = """
Priya Sharma
BCA Student, Semester 4
Gujarat, India
"""
print(bio)
```

## Accessing Characters — Indexing

Every character in a string has a position number starting from **0**:

```python
language = "Python"

#  P   y   t   h   o   n
#  0   1   2   3   4   5
# -6  -5  -4  -3  -2  -1

print(language[0])     # P
print(language[3])     # h
print(language[-1])    # n (last character)
print(language[-2])    # o (second last)
```

## Extracting Parts — Slicing

Slicing lets you grab a portion of a string using `[start:end]`. The start index is **included**, the end index is **excluded**.

```python
topic = "DataScience"

print(topic[0:4])     # Data
print(topic[4:])      # Science
print(topic[:4])      # Data
print(topic[-7:])     # Science
```

### Slicing with Step

```python
numbers = "0123456789"

print(numbers[::2])    # 02468  (every 2nd character)
print(numbers[1::2])   # 13579  (odd positions)
print(numbers[::-1])   # 9876543210  (reversed!)
```

Reversing a string with `[::-1]` is a very popular Python trick.

## Joining Strings — Concatenation

Use `+` to combine strings:

```python
first = "Raj"
last = "Patel"
full_name = first + " " + last
print(full_name)   # Raj Patel
```

Use `*` to repeat strings:

```python
border = "-" * 30
print(border)   # ------------------------------
```

## String Length

The `len()` function counts all characters including spaces:

```python
msg = "Hello World"
print(len(msg))    # 11
```

## Essential String Methods

Python provides many built-in methods to manipulate strings:

```python
text = "  hello, code anantam  "

print(text.strip())          # "hello, code anantam" (removes spaces)
print(text.strip().upper())  # "HELLO, CODE ANANTAM"
print(text.strip().lower())  # "hello, code anantam"
print(text.strip().title())  # "Hello, Code Anantam"
```

### Searching Inside Strings

```python
sentence = "Python is versatile and Python is popular"

print(sentence.find("Python"))       # 0 (first occurrence position)
print(sentence.count("Python"))      # 2 (how many times)
print(sentence.startswith("Python")) # True
print(sentence.endswith("popular"))  # True
```

### Replacing Text

```python
original = "I love Java"
updated = original.replace("Java", "Python")
print(updated)   # I love Python
```

### Splitting and Joining

```python
csv_data = "Raj,Priya,Arjun,Meera"
names = csv_data.split(",")
print(names)   # ['Raj', 'Priya', 'Arjun', 'Meera']

rejoined = " | ".join(names)
print(rejoined)   # Raj | Priya | Arjun | Meera
```

## Checking String Content

```python
pin = "4023"
print(pin.isdigit())      # True — all digits

word = "Python"
print(word.isalpha())     # True — all letters

mixed = "Hello World"
print(mixed.isalpha())    # False — space is not a letter
```

## Strings are Immutable

You **cannot** change a character in a string directly:

```python
name = "Python"
# name[0] = "J"   # This will raise an error!
```

Instead, create a new string:

```python
name = "Python"
new_name = "J" + name[1:]
print(new_name)   # Jython
```

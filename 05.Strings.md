# Strings in Python

## 1. Introduction

A string is used to store text in Python.

Anything written inside single `' '` or double `" "` quotes is a string.

Strings are used for:

- Names
- Messages
- Addresses
- Emails
- Any text data

Strings are one of the most common data types in Python.

## 2. Creating Strings

```python
name = "Vikram"
city = 'Jaipur'
msg = "Welcome to Code Anantam"
```

Single and double quotes both work the same way.

## 3. Printing Strings

```python
print("Hello World")
```

Printing a variable:

```python
name = "Pooja"
print(name)
```

## 4. String Indexing

Each character has a position starting from **0**.

```python
text = "Python"
print(text[0])   # P
print(text[1])   # y
print(text[5])   # n
```

Index positions:

```
P  y  t  h  o  n
0  1  2  3  4  5
```

### Negative Indexing

Count from the end:

- `-1` = last character
- `-2` = second last

```python
text = "Python"
print(text[-1])   # n
print(text[-2])   # o
```

## 5. String Slicing

Extract a part of the string.

```python
text = "Programming"
print(text[0:7])   # Program
```

- Start index is included
- End index is excluded

More examples:

```python
text = "Python"
print(text[:3])    # Pyt (from start)
print(text[2:])    # thon (to end)
```

## 6. String Concatenation

Joining strings using `+`:

```python
first = "Good"
second = "Morning"
print(first + " " + second)   # Good Morning
```

## 7. String Repetition

Repeat strings using `*`:

```python
print("Ha" * 3)   # HaHaHa
```

## 8. Length of a String

Use `len()` to count characters:

```python
text = "Python"
print(len(text))   # 6
```

## 9. Useful String Methods

```python
msg = "hello world"

print(msg.upper())       # HELLO WORLD
print(msg.lower())       # hello world
print(msg.title())       # Hello World
print(msg.replace("hello", "hi"))   # hi world
print(msg.count("l"))    # 3
```

These methods return a new string. The original stays the same.

## 10. Strings Cannot Be Changed

Strings are **immutable** — you cannot change a single character.

```python
text = "Python"
# text[0] = "J"   # This gives an error!
```

## 11. Common Mistakes

- Forgetting quotes:

```python
name = Raj      # Error!
name = "Raj"    # Correct
```

- Wrong index:

```python
text = "Hi"
print(text[5])   # Error! Index out of range
```

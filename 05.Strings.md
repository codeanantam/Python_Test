# Strings in Python

String matlab text. Jab bhi hum quotes mein kuch likhte hain, wo string hota hai.

## String Banana

Single ya double quotes — dono se ban jaata hai:

```python
name = "Arjun"
city = 'Vadodara'
```

Bahut lambi text ke liye triple quotes:

```python
paragraph = """Ye ek
bahut lambi
string hai"""
print(paragraph)
```

## Characters Ko Access Karna — Indexing

Har character ki ek position hoti hai. Position **0** se shuru hoti hai:

```python
word = "HELLO"

#  H   E   L   L   O
#  0   1   2   3   4

print(word[0])   # H
print(word[4])   # O
```

### Ulte se count karna (Negative Index)

```python
word = "HELLO"
print(word[-1])   # O  (aakhri)
print(word[-3])   # L  (teesra last se)
```

## String Ka Hissa Nikalna — Slicing

`[start:end]` se kuch part nikal sakte hain. Start included hai, end excluded:

```python
text = "Code Anantam"

print(text[0:4])    # Code
print(text[5:])     # Anantam
print(text[:4])     # Code
```

### String Ulta Karna

```python
name = "Python"
print(name[::-1])   # nohtyP
```

Ye ek bahut popular Python trick hai!

## Strings Jodna

`+` se do strings jud jaate hain:

```python
first = "Good"
second = "Night"
print(first + " " + second)   # Good Night
```

`*` se string repeat hota hai:

```python
laugh = "Ha" * 3
print(laugh)   # HaHaHa

line = "-" * 20
print(line)    # --------------------
```

## String Ki Length

`len()` se characters count hote hain:

```python
msg = "Code Anantam"
print(len(msg))   # 12 (space bhi count hota hai)
```

## String Methods

Python mein string ke saath kuch ready-made functions aate hain:

### Case Change

```python
text = "hello world"
print(text.upper())       # HELLO WORLD
print(text.lower())       # hello world
print(text.title())       # Hello World
print(text.capitalize())  # Hello world
```

### Dhoondhna aur Badalna

```python
msg = "I like Java"
print(msg.replace("Java", "Python"))   # I like Python
print(msg.find("Java"))                # 7 (position)
```

### Count Karna

```python
sentence = "banana"
print(sentence.count("a"))   # 3
```

### Extra Spaces Hatana

```python
text = "  Hello  "
print(text.strip())   # "Hello"
```

> **Note:** Ye sab methods nayi string return karte hain. Original string nahi badalta.

## Strings Badal Nahi Sakte (Immutable)

String ke ek character ko directly change nahi kar sakte:

```python
name = "Python"
# name[0] = "J"   → Error aayega!
```

Nayi string banana padta hai:

```python
name = "Python"
new_name = "J" + name[1:]
print(new_name)   # Jython
```

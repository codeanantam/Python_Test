# Operators in Python

## Building Blocks of Logic

Operators are symbols that tell Python to perform specific computations or comparisons. Whether you're calculating a bill, checking a password, or filtering data — operators are involved.

## Arithmetic Operators

Used for mathematical computations.

```python
a = 20
b = 7

print(a + b)     # 27  → Addition
print(a - b)     # 13  → Subtraction
print(a * b)     # 140 → Multiplication
print(a / b)     # 2.857... → Division (always float)
print(a // b)    # 2   → Floor Division
print(a % b)     # 6   → Modulus (remainder)
print(a ** b)    # Exponentiation
```

### Practical Use: Bill Calculator

```python
item_cost = 450
qty = 3
tax_percent = 18

subtotal = item_cost * qty
tax = subtotal * tax_percent / 100
grand_total = subtotal + tax

print(f"Subtotal: ₹{subtotal}")
print(f"Tax (18%): ₹{tax}")
print(f"Grand Total: ₹{grand_total}")
```

### Even or Odd Check

```python
number = 47
if number % 2 == 0:
    print("Even")
else:
    print("Odd")
```

## Comparison Operators

Return `True` or `False` — the foundation of decision-making.

```python
x = 15
y = 20

print(x == y)    # False → Equal to
print(x != y)    # True  → Not equal
print(x > y)     # False → Greater than
print(x < y)     # True  → Less than
print(x >= 15)   # True  → Greater or equal
print(y <= 19)   # False → Less or equal
```

Strings can also be compared alphabetically:

```python
print("apple" < "banana")    # True
print("cat" == "Cat")        # False (case matters)
```

## Assignment Operators

Shorthand operators for updating variables:

```python
points = 50

points += 10    # 60
points -= 5     # 55
points *= 2     # 110
points //= 3    # 36
points %= 10    # 6
print(points)
```

## Logical Operators

Combine multiple conditions into one expression.

### `and` — All must be True

```python
age = 22
has_license = True
if age >= 18 and has_license:
    print("You can drive")
```

### `or` — At least one must be True

```python
is_weekend = False
is_holiday = True
if is_weekend or is_holiday:
    print("No classes today!")
```

### `not` — Flips the value

```python
logged_in = False
if not logged_in:
    print("Please log in first")
```

### Combined Example

```python
marks = 85
attendance = 78

if marks >= 75 and attendance >= 75:
    print("Eligible for certificate")
elif marks >= 75 or attendance >= 75:
    print("Partially eligible")
else:
    print("Not eligible")
```

## Membership Operators

Check whether a value exists **in** a sequence:

```python
fruits = ["mango", "apple", "grape"]
print("apple" in fruits)       # True
print("banana" not in fruits)  # True

sentence = "Python is powerful"
print("power" in sentence)     # True
```

## Operator Precedence

| Priority | Operators |
|----------|-----------|
| Highest | `**` |
| | `*`, `/`, `//`, `%` |
| | `+`, `-` |
| | Comparisons |
| | `not` → `and` → `or` |

```python
result = 5 + 3 * 2 ** 2   # 17
result2 = (5 + 3) * 2 ** 2  # 32
```

Use **parentheses** to make intent explicit.

## Try It Yourself

1. Take a price and discount %, calculate final price
2. Check if a year is a leap year
3. Check if a character is a vowel using `in`

```python
# Discount Calculator
price = float(input("Enter price: "))
discount = float(input("Discount %: "))
final = price - (price * discount / 100)
print(f"You pay: ₹{final}")

# Vowel Checker
ch = input("Enter a character: ").lower()
if ch in "aeiou":
    print(f"'{ch}' is a vowel")
else:
    print(f"'{ch}' is a consonant")
```

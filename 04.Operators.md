# Operators in Python

Operators wo symbols hain jinse Python mein calculations, comparisons aur conditions likhte hain.

## Arithmetic — Math Ke Operators

```python
a = 18
b = 4

print(a + b)    # 22   Jodna
print(a - b)    # 14   Ghatana
print(a * b)    # 72   Gunna
print(a / b)    # 4.5  Bhaag (hamesha decimal aata hai)
print(a // b)   # 4    Bhaag bina decimal
print(a % b)    # 2    Baaki (remainder)
print(a ** b)   # Power: 18^4
```

### Real Example: Dukaan Ka Bill

```python
samosa_price = 15
samosa_qty = 4

chai_price = 10
chai_qty = 4

bill = (samosa_price * samosa_qty) + (chai_price * chai_qty)
print(f"Total bill: ₹{bill}")   # Total bill: ₹100
```

### Even ya Odd Pata Karo

`%` (remainder) operator se even/odd check hota hai:

```python
num = 7

if num % 2 == 0:
    print("Even hai")
else:
    print("Odd hai")
```

Agar remainder 0 hai → Even, warna Odd.

## Comparison — Do Values Compare Karo

Result hamesha `True` ya `False` aata hai:

```python
x = 10
y = 20

print(x == y)    # False  Barabar hai?
print(x != y)    # True   Alag hai?
print(x > y)     # False  Bada hai?
print(x < y)     # True   Chhota hai?
print(x >= 10)   # True   Bada ya barabar?
print(y <= 15)   # False  Chhota ya barabar?
```

## Logical — Conditions Jodo

### `and` — Dono sach hone chahiye

```python
marks = 75
attendance = 80

if marks >= 40 and attendance >= 75:
    print("Pass!")
```

### `or` — Ek bhi sach ho toh kaafi

```python
day = "Sunday"
if day == "Saturday" or day == "Sunday":
    print("Chutti hai!")
```

### `not` — Ulta kar do

```python
raining = False
if not raining:
    print("Bahar chalo!")
```

## Assignment — Value Update Karo

Normal assignment ke alaawa shortcut bhi hain:

```python
score = 100

score += 10    # score = 100 + 10 → 110
score -= 20    # score = 110 - 20 → 90
score *= 2     # score = 90 * 2  → 180

print(score)   # 180
```

## Membership — Andar Hai Ya Nahi?

`in` se check hota hai ki koi cheez list ya string mein hai ya nahi:

```python
colors = ["red", "blue", "green"]
print("blue" in colors)       # True
print("yellow" in colors)     # False
print("yellow" not in colors) # True
```

String mein bhi kaam karta hai:

```python
msg = "Python is fun"
print("fun" in msg)   # True
```

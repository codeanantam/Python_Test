# Operators in Python

## 1. Introduction

Operators are symbols that perform operations on values and variables.

For example:

- Adding two numbers
- Comparing values
- Checking conditions

Operators are used everywhere in programming to build logic.

## 2. Arithmetic Operators

Used for math calculations.

| Operator | Meaning |
|----------|---------|
| `+` | Addition |
| `-` | Subtraction |
| `*` | Multiplication |
| `/` | Division |
| `%` | Modulus (remainder) |
| `**` | Power |
| `//` | Floor Division |

### Addition

```python
a = 15
b = 5
print(a + b)   # 20
```

### Subtraction

```python
print(20 - 8)   # 12
```

### Multiplication

```python
print(6 * 3)   # 18
```

### Division

```python
print(15 / 2)   # 7.5
```

Division always gives a float result.

### Modulus

Gives the remainder after division.

```python
print(10 % 3)   # 1
```

Useful to check even/odd:

```python
num = 6
print(num % 2)   # 0 means even
```

### Power

```python
print(2 ** 3)   # 8
```

### Floor Division

Gives result without decimal.

```python
print(15 // 2)   # 7
```

## 3. Comparison Operators

Used to compare two values. Result is always `True` or `False`.

| Operator | Meaning |
|----------|---------|
| `==` | Equal to |
| `!=` | Not equal |
| `>` | Greater than |
| `<` | Less than |
| `>=` | Greater or equal |
| `<=` | Less or equal |

**Example:**

```python
print(10 > 5)     # True
print(5 == 5)     # True
print(3 != 7)     # True
print(8 < 2)      # False
```

These are mainly used in conditions.

## 4. Logical Operators

Used to combine conditions.

| Operator | Meaning |
|----------|---------|
| `and` | Both must be True |
| `or` | At least one must be True |
| `not` | Reverses the result |

### AND

```python
age = 22
print(age > 18 and age < 30)   # True
```

Both conditions must be true.

### OR

```python
print(5 > 10 or 3 < 8)   # True
```

At least one is true.

### NOT

```python
print(not True)    # False
print(not False)   # True
```

## 5. Assignment Operators

Shorthand for updating variables.

```python
x = 10
x += 5     # x = x + 5 → 15
x -= 3     # x = x - 3 → 12
x *= 2     # x = x * 2 → 24
print(x)
```

## 6. Common Mistakes

- Using `=` instead of `==` when comparing
- Forgetting to convert strings to numbers before math
- Mixing up `%` with `/`

**Example mistake:**

```python
age = "25"
print(age + 5)   # Error!
```

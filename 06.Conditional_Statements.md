# Conditional Statements

## Decisions Lena Code Mein

Real life mein hum hamesha decisions lete hain:

- Agar baarish ho rahi hai → chhatri le jao
- Agar marks 40 se zyada hain → pass

Python mein aisi decisions `if`, `elif` aur `else` se likhte hain.

## `if` — Agar Ye Sach Hai Toh

```python
temperature = 42

if temperature > 40:
    print("Bahut garmi hai! Paani piyo")
```

Agar condition `True` hai toh code chalega. `False` hai toh skip.

## Indentation Zaroori Hai

Python mein `if` ke andar ka code **4 spaces** se aage likhna hota hai:

```python
# Sahi ✅
if True:
    print("Andar")

# Galat ❌
if True:
print("Andar")   # Error!
```

## `if-else` — Do Raaste

```python
balance = 500
withdraw = 800

if withdraw <= balance:
    print("Paisa nikal gaya")
else:
    print("Balance kam hai!")
```

## `if-elif-else` — Bahut Options

```python
marks = 68

if marks >= 90:
    print("A Grade")
elif marks >= 75:
    print("B Grade")
elif marks >= 60:
    print("C Grade")
elif marks >= 40:
    print("D Grade")
else:
    print("Fail")
```

## Nested `if`

```python
age = 20
has_ticket = True

if age >= 18:
    if has_ticket:
        print("Movie dekh sakte ho")
    else:
        print("Pehle ticket lo")
```

## Logical Operators

### `and` — dono sach ho

```python
username = "admin"
password = "1234"

if username == "admin" and password == "1234":
    print("Login successful!")
```

### `or` — koi ek sach ho

```python
day = "Saturday"
if day == "Saturday" or day == "Sunday":
    print("Weekend hai!")
```

### `not` — ulta karo

```python
banned = False
if not banned:
    print("Welcome!")
```

## One-Liner (Ternary)

```python
age = 16
result = "Adult" if age >= 18 else "Minor"
print(result)   # Minor
```

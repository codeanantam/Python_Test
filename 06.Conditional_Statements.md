# Conditional Statements in Python

**(if, if-else, if-elif-else)**

## 1. Introduction

In programming, we often need to make decisions.

Examples:

- If marks are above 40 → Pass
- If age is 18 or more → Can vote
- If number is even → Print "Even"

**Conditional statements** allow the program to choose what to do based on a condition.

## 2. The `if` Statement

Runs the code only if the condition is `True`.

```python
age = 20

if age >= 18:
    print("You can vote")
```

If the condition is false, nothing happens.

## 3. Indentation is Important

Python uses **spaces** to define blocks of code.

**Correct:**

```python
if 10 > 5:
    print("Yes")
```

**Wrong:**

```python
if 10 > 5:
print("Yes")   # Error!
```

Always indent with 4 spaces inside `if`.

## 4. The `if-else` Statement

Two choices — one for True, one for False.

```python
num = 9

if num % 2 == 0:
    print("Even")
else:
    print("Odd")
```

## 5. The `if-elif-else` Statement

For multiple conditions.

```python
marks = 78

if marks >= 90:
    print("Grade A")
elif marks >= 75:
    print("Grade B")
elif marks >= 60:
    print("Grade C")
else:
    print("Grade D")
```

Python checks from top to bottom and stops at the first true condition.

## 6. Nested `if`

An `if` inside another `if`.

```python
age = 20
has_id = True

if age >= 18:
    if has_id:
        print("Entry allowed")
    else:
        print("Bring your ID")
```

## 7. Using Logical Operators

Combine multiple conditions:

```python
age = 25
has_ticket = True

if age >= 18 and has_ticket:
    print("You can enter")
```

```python
day = "Sunday"

if day == "Saturday" or day == "Sunday":
    print("It's weekend!")
```

```python
is_blocked = False

if not is_blocked:
    print("Access granted")
```

## 8. Short Form – Ternary Expression

For simple conditions, use one-line format:

```python
age = 16
status = "Adult" if age >= 18 else "Minor"
print(status)   # Minor
```

## 9. Common Mistakes

- Using `=` instead of `==`:

```python
if age = 18:    # Wrong!
if age == 18:   # Correct
```

- Forgetting the colon `:` after condition
- Wrong indentation

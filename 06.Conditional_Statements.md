# Conditional Statements

## Making Decisions in Code

Programs often need to choose between different actions based on some condition. Should we grant access? Is the student passing? Is the input valid? **Conditional statements** let your program make these decisions.

## The `if` Statement

The simplest form — execute code only when a condition is `True`.

```python
temperature = 40

if temperature > 35:
    print("It's very hot outside! Stay hydrated.")
```

If the condition evaluates to `False`, the code block is simply skipped.

### Indentation Matters

Python uses **indentation** (typically 4 spaces) to define what belongs inside a block. This is not optional — it's part of the syntax.

```python
# Correct
if True:
    print("Inside the block")
    print("Still inside")

print("Outside the block")
```

## The `if-else` Statement

When you need two branches — one for `True` and one for `False`:

```python
balance = 2500
withdrawal = 3000

if withdrawal <= balance:
    balance -= withdrawal
    print(f"Withdrawn ₹{withdrawal}. Remaining: ₹{balance}")
else:
    print("Insufficient balance!")
```

## The `if-elif-else` Ladder

For multiple possible conditions, use `elif` (short for "else if"):

```python
percentage = 72

if percentage >= 90:
    grade = "A+"
elif percentage >= 80:
    grade = "A"
elif percentage >= 70:
    grade = "B"
elif percentage >= 60:
    grade = "C"
elif percentage >= 40:
    grade = "D"
else:
    grade = "Fail"

print(f"Your grade: {grade}")
```

Python checks conditions **from top to bottom** and stops at the first match.

## Nested Conditions

You can place `if` statements inside other `if` statements:

```python
applicant_age = 25
has_degree = True
experience = 3

if applicant_age >= 21:
    if has_degree:
        if experience >= 2:
            print("Shortlisted for interview")
        else:
            print("Need more experience")
    else:
        print("Degree required")
else:
    print("Age requirement not met")
```

> **Tip:** Deeply nested code can become hard to read. Consider combining conditions with `and`/`or` when possible.

## Using Logical Operators

Combine multiple conditions in a single `if`:

```python
username = "admin"
password = "secure123"

if username == "admin" and password == "secure123":
    print("Login successful!")
else:
    print("Invalid credentials")
```

```python
day = "Sunday"

if day == "Saturday" or day == "Sunday":
    print("Weekend — enjoy your day off!")
else:
    print("Weekday — time to work!")
```

```python
is_banned = False

if not is_banned:
    print("Welcome to the platform")
```

## Ternary Expression (One-liner)

For simple conditions, Python offers a compact syntax:

```python
age = 17
status = "Adult" if age >= 18 else "Minor"
print(status)   # Minor
```

Another example:

```python
score = 85
result = "Pass" if score >= 40 else "Fail"
print(result)   # Pass
```

## Real-World Example: Ticket Pricing

```python
age = int(input("Enter your age: "))

if age < 5:
    ticket = "Free"
    price = 0
elif age <= 12:
    ticket = "Child"
    price = 100
elif age <= 59:
    ticket = "Adult"
    price = 250
else:
    ticket = "Senior Citizen"
    price = 150

print(f"Ticket type: {ticket}")
print(f"Price: ₹{price}")
```

## Try It Yourself

1. Take a number and check if it's positive, negative, or zero
2. Build a simple calculator: take two numbers and an operator (+, -, *, /), then show the result
3. Take three numbers and print the largest one

```python
# Simple Calculator
n1 = float(input("First number: "))
op = input("Operator (+, -, *, /): ")
n2 = float(input("Second number: "))

if op == "+":
    print(f"Result: {n1 + n2}")
elif op == "-":
    print(f"Result: {n1 - n2}")
elif op == "*":
    print(f"Result: {n1 * n2}")
elif op == "/":
    if n2 != 0:
        print(f"Result: {n1 / n2}")
    else:
        print("Cannot divide by zero!")
else:
    print("Invalid operator")
```

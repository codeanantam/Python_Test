# Type Casting and Input/Output in Python

## 1. Introduction

To make programs useful, we need to:

- Show results to users
- Take input from users
- Convert data from one type to another

These are the basics of making interactive programs.

## 2. Output in Python – `print()`

The `print()` function shows output on screen.

**Basic example:**

```python
print("Welcome to Code Anantam")
print(200)
```

### Printing Variables

```python
name = "Rohit"
age = 22
print(name)
print(age)
```

### Printing Multiple Values

```python
city = "Pune"
year = 2026
print(city, year)
```

Python adds a space between values automatically.

### Using f-Strings

F-strings let you put variables inside text easily.

```python
name = "Sneha"
age = 21
print(f"My name is {name} and I am {age} years old.")
```

This is the best way to format output.

## 3. Input in Python – `input()`

The `input()` function lets the user type something.

**Example:**

```python
name = input("What is your name? ")
print("Hello", name)
```

### Important: Input is Always a String

Even if the user types a number, Python treats it as text.

```python
age = input("Enter your age: ")
print(type(age))
```

**Output:**

```
<class 'str'>
```

That's why we need type casting.

## 4. Type Casting

Type casting means changing one data type to another.

### Convert to Integer

```python
age = int(input("Enter your age: "))
print(type(age))   # <class 'int'>
```

### Convert to Float

```python
price = float(input("Enter price: "))
print(type(price))   # <class 'float'>
```

### Convert to String

```python
num = 50
text = str(num)
print(type(text))   # <class 'str'>
```

## 5. Why Type Casting Matters

Look at this example:

```python
a = input("Enter first number: ")
b = input("Enter second number: ")
print(a + b)
```

If user enters `10` and `20`:

**Output:** `1020` (because they are strings!)

**Correct way:**

```python
a = int(input("Enter first number: "))
b = int(input("Enter second number: "))
print(a + b)
```

**Output:** `30`

## 6. Common Mistakes

- Not converting input before doing math
- Trying to add a string and number together
- Writing `Int()` instead of `int()` (Python is case-sensitive)

# Type Casting and Input/Output

## Making Programs Interactive

A program that only works with hardcoded values isn't very useful. Real applications need to **accept input** from users, **process** that data, and **display** meaningful output. This chapter covers exactly how to do that in Python.

## Displaying Output with `print()`

The `print()` function sends text or values to the console.

### Basic Usage

```python
print("Learning Python at Code Anantam")
print(2026)
print(3.14)
```

### Printing Variables

```python
course = "Python Fundamentals"
duration = 30
print(course)
print(duration)
```

### Combining Text and Variables

**Method 1: Using commas (simplest)**

```python
brand = "Tesla"
year = 2025
print("Company:", brand, "| Founded near:", year)
```

Python inserts a space between comma-separated values automatically.

**Method 2: Using f-strings (recommended)**

```python
item = "Keyboard"
cost = 1500
print(f"The {item} costs ₹{cost}")
```

**Output:**

```
The Keyboard costs ₹1500
```

F-strings are the most modern and readable way to format output. You can even put expressions inside:

```python
length = 12
width = 8
print(f"Area of rectangle: {length * width} sq units")
```

**Method 3: String concatenation with `+`**

```python
first = "Code"
second = "Anantam"
print(first + " " + second)
```

> **Warning:** You can only concatenate strings with strings. `"Age: " + 25` will cause an error. Use `str(25)` to convert first.

### Customizing `print()` Behavior

```python
# Change separator
print("Python", "Java", "Go", sep=" → ")
# Output: Python → Java → Go

# Change ending character
print("Loading", end="...")
print("Done")
# Output: Loading...Done
```

## Accepting Input with `input()`

The `input()` function pauses the program and waits for the user to type something.

```python
city = input("Which city are you from? ")
print(f"Nice! {city} is a great place.")
```

### Critical Rule: `input()` Always Returns a String

No matter what the user types — even if it looks like a number — Python treats it as text.

```python
birth_year = input("Enter your birth year: ")
print(type(birth_year))
```

**Output:**

```
<class 'str'>
```

This means you **cannot** do math directly with input values. You must convert them first.

## Type Casting — Converting Between Types

Type casting is the process of changing a value from one data type to another.

### Available Casting Functions

| Function | Converts To | Example |
|----------|-------------|---------|
| `int()` | Integer | `int("42")` → `42` |
| `float()` | Decimal | `float("9.8")` → `9.8` |
| `str()` | String | `str(100)` → `"100"` |
| `bool()` | Boolean | `bool(1)` → `True` |

### Practical Examples

**Taking numeric input:**

```python
quantity = int(input("How many items? "))
unit_price = float(input("Price per item: "))

total = quantity * unit_price
print(f"Total bill: ₹{total}")
```

**Converting for display:**

```python
marks = 92
message = "You scored " + str(marks) + " marks!"
print(message)
```

### The Classic Input Trap

```python
a = input("First number: ")
b = input("Second number: ")
print(a + b)
```

If the user enters `15` and `25`, the output is `1525` (string concatenation), not `40`.

**Fix:**

```python
a = int(input("First number: "))
b = int(input("Second number: "))
print(f"Sum = {a + b}")
```

## Understanding Boolean Casting

Python considers certain values as "truthy" or "falsy":

```python
print(bool(0))       # False
print(bool(""))      # False
print(bool(None))    # False

print(bool(1))       # True
print(bool(-5))      # True
print(bool("text"))  # True
```

**Rule of thumb:** Empty or zero values are `False`, everything else is `True`.

## Handling Invalid Input

What happens if a user types "hello" when you expect a number?

```python
num = int(input("Enter a number: "))
# If user types "hello" → ValueError!
```

For now, just be aware of this. We'll learn how to handle such errors gracefully using `try-except` in a later chapter.

## Try It Yourself

1. Ask the user for their name and birth year, then calculate and print their approximate age
2. Take the length and width of a room as input, compute the area
3. Ask for a temperature in Celsius and convert it to Fahrenheit using: `F = (C × 9/5) + 32`

```python
# Age Calculator
name = input("Your name: ")
birth = int(input("Birth year: "))
current_year = 2026

age = current_year - birth
print(f"{name}, you are approximately {age} years old.")

# Temperature Converter
celsius = float(input("Temperature in Celsius: "))
fahrenheit = (celsius * 9/5) + 32
print(f"{celsius}°C = {fahrenheit}°F")
```

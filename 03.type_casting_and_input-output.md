# Type Casting aur Input/Output

## Output Dikhana — `print()`

Sabse pehle seekhte hain screen pe kuch dikhana:

```python
print("Namaste Duniya!")
print(999)
```

### Variables ko print karna

```python
subject = "Python"
chapter = 3
print(subject)
print(chapter)
```

### Saath mein print karna

**Comma use karke:**

```python
fruit = "Mango"
qty = 5
print(fruit, qty)   # Mango 5
```

**f-string se (sabse acha tarika):**

```python
item = "Notebook"
price = 40
print(f"{item} ki price {price} rupees hai")
```

Output: `Notebook ki price 40 rupees hai`

F-string mein `{}` ke andar variable ya calculation likh sakte ho:

```python
l = 10
b = 5
print(f"Area = {l * b}")   # Area = 50
```

## User Se Input Lena — `input()`

`input()` se program ruk jaata hai aur user se value maangta hai:

```python
naam = input("Apna naam batao: ")
print("Hello", naam)
```

Jab ye chalega, screen pe aayega:

```
Apna naam batao: Raj
Hello Raj
```

### Zaroori Baat: Input Hamesha String Hota Hai

Chahe user number bhi type kare, Python use text maanta hai:

```python
num = input("Number daalo: ")
print(type(num))   # <class 'str'>
```

Isliye agar math karni ho, toh pehle convert karna padega.

## Type Casting — Type Badalna

Type casting ka matlab hai value ko ek type se doosre type mein convert karna.

### `int()` — Integer mein badlo

```python
age = int(input("Age daalo: "))
print(age + 1)   # Ab math ho sakti hai
```

### `float()` — Decimal mein badlo

```python
weight = float(input("Weight daalo: "))
print(weight)
```

### `str()` — String mein badlo

```python
roll = 101
msg = "Roll number: " + str(roll)
print(msg)
```

## Bina Casting Ke Kya Hoga?

```python
a = input("Pehla number: ")
b = input("Doosra number: ")
print(a + b)
```

User ne daala `5` aur `3`:

Output: `53` ← Ye text jud gaya, math nahi hui!

**Sahi tarika:**

```python
a = int(input("Pehla number: "))
b = int(input("Doosra number: "))
print("Jod:", a + b)   # Jod: 8
```

## Mini Project: Student Info Card

```python
name = input("Student ka naam: ")
age = int(input("Age: "))
marks = float(input("Marks: "))

print("\n--- Student Card ---")
print(f"Naam   : {name}")
print(f"Age    : {age}")
print(f"Marks  : {marks}")
print("--------------------")
```

**Output:**

```
Student ka naam: Meera
Age: 20
Marks: 87.5

--- Student Card ---
Naam   : Meera
Age    : 20
Marks  : 87.5
--------------------
```

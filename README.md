# Python Syllabus for Code Anantam Interview Round

## Syllabus (Till Loops)

### 1. Course Overview

This course is built for absolute beginners who have never written a single line of code before. It covers the essential building blocks of Python programming.

By the end of this syllabus, students will be able to:

- Understand how programming works
- Write and run Python programs independently
- Accept input from users and display results
- Use operators for calculations and comparisons
- Make decisions in code using conditions
- Automate repetitive tasks using loops

This syllabus focuses on core fundamentals that are necessary before moving into advanced topics such as Lists, Functions, File Handling, and Object-Oriented Programming.

### 2. Learning Goals

After completing this syllabus, students will be able to:

- Read and write Python code confidently
- Create and use variables with different data types
- Handle user input and format output properly
- Perform arithmetic, comparison, and logical operations
- Build decision-making logic using if-elif-else
- Write loops to solve repetition-based problems

### 3. Topic Breakdown

#### Module 1: Introduction to Python
- What is Python and why is it popular?
- Key features of Python
- Installing Python on your system
- Running your first Python program
- Using the `print()` function

#### Module 2: Variables and Basic Syntax
- Understanding indentation
- Writing comments using `#`
- Python keywords (overview)
- Creating variables
- Variable naming rules
- Assigning and reassigning values
- Dynamic typing in Python

#### Module 3: Data Types
- Integer (`int`)
- Float (`float`)
- String (`str`)
- Boolean (`bool`)
- Checking types using `type()`

#### Module 4: Type Casting
- Converting between types using:
  - `int()`
  - `float()`
  - `str()`

#### Module 5: Input and Output
- Displaying output with `print()`
- Taking user input with `input()`
- Handling user input properly
- Formatting output using f-strings

#### Module 6: Operators
- **Arithmetic Operators:** `+`, `-`, `*`, `/`, `%`, `//`, `**`
- **Comparison Operators:** `==`, `!=`, `>`, `<`, `>=`, `<=`
- **Logical Operators:** `and`, `or`, `not`
- **Assignment Operators:** `+=`, `-=`, `*=`

#### Module 7: Strings
- Creating strings (single & double quotes)
- Accessing characters using indexing
- Extracting parts using slicing
- Joining strings with concatenation
- Common string methods

#### Module 8: Conditional Statements
- `if` statement
- `if-else` statement
- `if-elif-else` ladder
- Nested conditions (basic level)
- Using logical operators in conditions

#### Module 9: Loops
- **For Loop**
  - Using `range()` function
  - Looping through numbers
  - Looping through strings
- **While Loop**
  - Condition-based repetition
  - Avoiding infinite loops
- **Loop Control**
  - `break` — stop the loop
  - `continue` — skip current iteration

### 4. Recommended Duration

This syllabus is designed to be mobile-learning friendly:

- **Duration:** 4 to 7 days
- **Daily Effort:** 1 to 3 hours per day

### 5. What Comes Next?

After completing this syllabus, students will be ready to learn:

- Lists and Tuples
- Functions
- Pattern printing problems
- Basic problem solving
- Mini console-based projects

# Introduction to Python

Python ek programming language hai jo 1991 mein Guido van Rossum ne banayi thi. Aaj ye duniya ki sabse popular languages mein se ek hai.

## Python Kahan Use Hota Hai?

Python bahut jagah kaam aata hai:

- Websites banana (Django, Flask)
- Data ko analyze karna (Pandas, NumPy)
- AI aur Machine Learning (TensorFlow)
- Daily tasks automate karna
- Games banana

## Python Kyun Seekhein?

Python ka code bilkul English jaisa hota hai. Dekhiye farak:

**Java mein "Hello" print karna:**
```java
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello");
    }
}
```

**Python mein same kaam:**
```python
print("Hello")
```

Bas ek line! Isliye beginners ke liye Python sabse acha hai.

## Python Install Karna

1. [python.org](https://www.python.org) pe jaayein
2. Latest version download karein
3. Install karein — Windows pe "Add to PATH" check karein
4. Terminal mein verify karein:

```
python --version
```

## Apna Pehla Program

Ek file banao `app.py` naam se:

```python
print("Code Anantam mein aapka swagat hai!")
```

Terminal mein chalao:

```
python app.py
```

Screen pe dikhega:

```
Code Anantam mein aapka swagat hai!
```

## `print()` Function

`print()` se hum screen pe kuch bhi dikha sakte hain:

```python
print("Namaste")
print(42)
print(3.14)
print(True)
```

Ek saath multiple cheezein print karna:

```python
print("Mera naam", "Arjun", "hai")
```

Output: `Mera naam Arjun hai`

## Comments — Code Mein Notes

Kabhi kabhi hum code mein notes likhte hain apne liye. Python inhe ignore karta hai.

```python
# Ye ek comment hai
print("Hello")   # Ye bhi comment hai
```

Multiple lines ka comment:

```python
"""
Ye program
sirf practice ke liye hai
"""
print("Practice!")
```

# Getting Started with Python

## What is Python?

Python is one of the most widely adopted programming languages in the world today. Created by Guido van Rossum in 1991, it was designed with a focus on code readability and simplicity.

Python powers a wide range of applications:

- **Backend Web Development** – Frameworks like Django and Flask
- **Data Science & Analytics** – Libraries like Pandas, NumPy
- **Artificial Intelligence** – TensorFlow, PyTorch, Scikit-learn
- **Scripting & Automation** – Automate repetitive tasks
- **Desktop Applications** – Using Tkinter, PyQt

### Why Choose Python as Your First Language?

Unlike languages such as Java or C++, Python removes unnecessary complexity. There are no semicolons to end statements, no curly braces for blocks, and no need to declare variable types.

Compare printing a message in different languages:

**C++:**
```cpp
#include <iostream>
using namespace std;
int main() {
    cout << "Hello" << endl;
    return 0;
}
```

**Python:**
```python
print("Hello")
```

The difference is clear — Python lets you focus on solving problems rather than writing boilerplate code.

## How Does Python Execute Code?

Python is an **interpreted language**. This means:

1. You write your code in a `.py` file
2. The Python interpreter reads it **line by line**
3. Each line is executed immediately

This is different from **compiled languages** (like C or Java) where the entire program must be compiled before running. The interpreted approach makes debugging faster since you see errors as soon as they occur.

## Setting Up Your Environment

### Step 1: Download Python

Head to [python.org/downloads](https://www.python.org/downloads/) and grab the latest version (Python 3.x).

> **Important:** During installation on Windows, make sure to check **"Add Python to PATH"**. This saves you from configuration headaches later.

### Step 2: Verify Installation

Open your terminal (Command Prompt on Windows, Terminal on Mac/Linux) and type:

```bash
python --version
```

You should see something like:

```
Python 3.12.x
```

On some systems (especially Mac/Linux), you may need to use `python3` instead.

## Writing Your First Program

Create a new file called `welcome.py` and write:

```python
print("Welcome to Code Anantam!")
```

To run it, open your terminal and execute:

```bash
python welcome.py
```

**Output:**

```
Welcome to Code Anantam!
```

### Breaking Down the Code

- `print()` — A built-in function that displays text on the screen
- `"Welcome to Code Anantam!"` — This is a **string** (text data enclosed in quotes)
- Python executes this single line and outputs the result immediately

## Python Interactive Mode (REPL)

You can also run Python commands directly without creating a file. Open your terminal and type `python`:

```
>>> 5 + 3
8
>>> print("Quick test")
Quick test
>>> 2 ** 10
1024
```

This is called the **REPL** (Read-Eval-Print Loop). It's great for quick calculations and testing small code snippets.

Type `exit()` to leave the interactive mode.

## Comments in Python

Comments help you explain your code. Python ignores them during execution.

**Single-line comment:**

```python
# This calculates the total price
total = 500 + 200
```

**Multi-line comment:**

```python
"""
This program calculates
the final bill amount
including tax
"""
bill = 1500
```

Writing clear comments is a habit that separates good programmers from great ones.

# Introduction to Python

## 1. What is Python?

Python is a high-level, interpreted programming language. It is used in many fields:

- Web development
- Automation
- Data analysis
- Machine learning
- Game development

Python is known for its simple and clean syntax, which makes it perfect for beginners.

**Example:**

```python
print("Hello World")
```

## 2. Why Python is Great for Beginners

Python is easy to learn because:

- No curly braces `{}` needed
- No semicolons `;` at the end of lines
- Code reads like English
- Less code to write compared to other languages

## 3. How Python Works

Python is an **interpreted** language. This means it reads and runs your code **line by line**. If there is an error, it stops and tells you exactly where the problem is.

## 4. Installing Python

1. Go to [https://www.python.org](https://www.python.org)
2. Download the latest version
3. Install it (check "Add to PATH" on Windows)
4. Verify by typing in terminal:

```
python --version
```

## 5. Where to Write Python Code

You can write Python in:

- **IDLE** – comes with Python
- **VS Code** – popular code editor
- **Terminal / Command Prompt**
- **Online editors** – like replit.com

## 6. Running a Python Program

Create a file called `hello.py`:

```python
print("Hello from Code Anantam!")
```

Run it in terminal:

```
python hello.py
```

## 7. Understanding the First Program

```python
print("Hello World")
```

- `print()` is a built-in function that shows output
- `"Hello World"` is a string (text)
- The output appears on the screen

## 8. Comments in Python

Comments are notes in your code. Python ignores them.

```python
# This is a comment
print("Hi")  # This prints Hi
```

# Loops in Python

**(for Loop and while Loop)**

## 1. Introduction

Sometimes we need to repeat a task multiple times.

Examples:

- Print numbers from 1 to 10
- Print a message 5 times
- Check numbers one by one
- Process multiple inputs

Instead of writing the same code again and again, we use **loops**.

Loops allow us to execute a block of code repeatedly.

There are two main loops in Python:

- `for` loop
- `while` loop

## 2. The `for` Loop

The `for` loop is mainly used when we know how many times we want to repeat something.

### Using `range()`

`range()` is commonly used with `for` loop.

**Example 1: Print 1 to 5**

```python
for i in range(1, 6):
    print(i)
```

**Output:**

```
1
2
3
4
5
```

**Important:**

- Start value is included
- End value is excluded

### Different Forms of `range()`

**`range(stop)`**

```python
for i in range(5):
    print(i)
```

**Output:**

```
0
1
2
3
4
```

**`range(start, stop)`**

```python
for i in range(2, 6):
    print(i)
```

**`range(start, stop, step)`**

```python
for i in range(1, 10, 2):
    print(i)
```

**Output:**

```
1
3
5
7
9
```

### Looping Through a String

```python
text = "Python"

for ch in text:
    print(ch)
```

This prints each character one by one.

## 3. The `while` Loop

The `while` loop runs as long as the condition is `True`.

**Syntax:**

```python
while condition:
    statement
```

**Example: Print 1 to 5**

```python
i = 1

while i <= 5:
    print(i)
    i += 1
```

**Important:**

- Always update the variable inside the loop
- Otherwise, it becomes an **infinite loop**

### Infinite Loop Example (Wrong)

```python
i = 1

while i <= 5:
    print(i)
```

This will run forever because `i` never changes.

## 4. Loop Control Statements

### `break`

Stops the loop completely.

```python
for i in range(1, 10):
    if i == 5:
        break
    print(i)
```

**Output:**

```
1
2
3
4
```

### `continue`

Skips the current iteration.

```python
for i in range(1, 6):
    if i == 3:
        continue
    print(i)
```

**Output:**

```
1
2
4
5
```

## 5. The `else` Clause with Loops

Python allows an `else` block after a loop. The `else` block runs only when the loop completes **without** hitting a `break`.

```python
for i in range(1, 6):
    print(i)
else:
    print("Loop completed successfully!")
```

**Output:**

```
1
2
3
4
5
Loop completed successfully!
```

If a `break` is triggered, the `else` block is skipped:

```python
for i in range(1, 6):
    if i == 3:
        break
    print(i)
else:
    print("This will NOT print")
```

## 6. Common Beginner Mistakes

- Forgetting to update variable in `while` loop
- Wrong indentation
- Using incorrect range values
- Creating accidental infinite loops

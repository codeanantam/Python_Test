# Loops in Python

## Kaam Ko Baar Baar Karna

Socho tumhe 1 se 100 tak numbers print karne hain. Kya 100 baar `print()` likhoge? Nahi!

Iske liye **loops** hain — ek kaam baar baar repeat karna.

Python mein do tarah ke loops hain:

- **`for` loop** — jab pata ho kitni baar chalana hai
- **`while` loop** — jab condition pe depend kare

## `for` Loop

### `range()` Ke Saath

`range()` numbers ki sequence deta hai:

**1 se 5 print karo:**

```python
for i in range(1, 6):
    print(i)
```

Output: `1 2 3 4 5`

> Yaad rakho: range(1, 6) mein 1 included hai, 6 excluded.

### `range()` Ke Alag Tarike

```python
# 0 se start (sirf end batao)
for i in range(4):
    print(i)   # 0, 1, 2, 3

# Step ke saath (2-2 karke)
for i in range(0, 11, 2):
    print(i)   # 0, 2, 4, 6, 8, 10

# Ulta gino
for i in range(5, 0, -1):
    print(i)   # 5, 4, 3, 2, 1
```

### String Pe Loop

```python
name = "PYTHON"
for letter in name:
    print(letter)
```

Har character ek-ek karke print hoga.

### Mini Project: Table Print Karo

```python
num = int(input("Kis number ka table chahiye? "))

for i in range(1, 11):
    print(f"{num} x {i} = {num * i}")
```

Output (agar user `7` daale):

```
7 x 1 = 7
7 x 2 = 14
...
7 x 10 = 70
```

## `while` Loop

Jab tak condition sach hai, tab tak chalta rahega:

```python
count = 1

while count <= 5:
    print(f"Attempt {count}")
    count += 1

print("Done!")
```

Output:

```
Attempt 1
Attempt 2
Attempt 3
Attempt 4
Attempt 5
Done!
```

### Password Checker

```python
secret = "python"

guess = input("Password daalo: ")

while guess != secret:
    print("Galat! Phir try karo")
    guess = input("Password daalo: ")

print("Sahi password! Welcome!")
```

## Loop Control

### `break` — Loop Band Karo

```python
for i in range(1, 20):
    if i == 6:
        print("6 mil gaya, ruk jaao!")
        break
    print(i)
```

### `continue` — Skip Karo, Aage Badho

```python
for i in range(1, 8):
    if i == 4:
        continue   # 4 skip hoga
    print(i)
```

Output: `1 2 3 5 6 7`

## Nested Loops — Loop Ke Andar Loop

```python
for i in range(1, 4):
    for j in range(1, 4):
        print(f"({i},{j})", end=" ")
    print()
```

Output:

```
(1,1) (1,2) (1,3)
(2,1) (2,2) (2,3)
(3,1) (3,2) (3,3)
```

### Star Pattern

```python
for i in range(1, 6):
    print("* " * i)
```

Output:

```
* 
* * 
* * * 
* * * * 
* * * * * 
```

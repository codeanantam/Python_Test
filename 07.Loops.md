# Loops in Python

## Automating Repetition

Imagine you need to print a message 100 times, or process every student's marks in a class. Writing the same code over and over is impractical. **Loops** solve this by letting you repeat a block of code automatically.

Python offers two types of loops:

- **`for` loop** — when you know how many iterations you need
- **`while` loop** — when repetition depends on a condition

## The `for` Loop

### Iterating with `range()`

`range()` generates a sequence of numbers for the loop to iterate through.

**Print numbers 1 through 5:**

```python
for num in range(1, 6):
    print(num)
```

**Output:**
```
1
2
3
4
5
```

> Remember: `range(start, stop)` — start is included, stop is **excluded**.

### `range()` Variations

```python
# Only stop value — starts from 0
for i in range(4):
    print(i)    # 0, 1, 2, 3

# With step — skip values
for i in range(0, 20, 5):
    print(i)    # 0, 5, 10, 15

# Counting backwards
for i in range(10, 0, -1):
    print(i)    # 10, 9, 8, ... 1
```

### Looping Through Strings

```python
word = "Code"
for letter in word:
    print(letter)
```

**Output:**
```
C
o
d
e
```

### Practical Example: Multiplication Table

```python
table_of = int(input("Enter a number: "))

print(f"\nMultiplication Table of {table_of}:")
print("-" * 20)

for i in range(1, 11):
    result = table_of * i
    print(f"{table_of} x {i} = {result}")
```

## The `while` Loop

Keeps running **as long as** the condition remains `True`.

```python
countdown = 5

while countdown > 0:
    print(f"Launching in {countdown}...")
    countdown -= 1

print("🚀 Launched!")
```

**Output:**
```
Launching in 5...
Launching in 4...
Launching in 3...
Launching in 2...
Launching in 1...
🚀 Launched!
```

### User Input Loop

A common pattern — keep asking until valid input is received:

```python
password = ""

while password != "python123":
    password = input("Enter password: ")

print("Access granted!")
```

> **Caution:** Always ensure the condition will eventually become `False`. Otherwise, you create an **infinite loop** that never stops.

### Infinite Loop (Bug!)

```python
# DON'T DO THIS
x = 1
while x <= 5:
    print(x)
    # Forgot x += 1 → runs forever!
```

If you accidentally run an infinite loop, press `Ctrl + C` to stop it.

## Loop Control: `break` and `continue`

### `break` — Exit the loop immediately

```python
for num in range(1, 100):
    if num == 8:
        print("Found 8! Stopping.")
        break
    print(num)
```

**Output:**
```
1
2
3
4
5
6
7
Found 8! Stopping.
```

### `continue` — Skip current iteration, move to next

```python
for num in range(1, 11):
    if num % 3 == 0:
        continue    # skip multiples of 3
    print(num)
```

**Output:**
```
1
2
4
5
7
8
10
```

## Nested Loops

A loop inside another loop. The inner loop completes all its iterations for each single iteration of the outer loop.

```python
for row in range(1, 4):
    for col in range(1, 4):
        print(f"({row},{col})", end="  ")
    print()   # new line after each row
```

**Output:**
```
(1,1)  (1,2)  (1,3)
(2,1)  (2,2)  (2,3)
(3,1)  (3,2)  (3,3)
```

### Pattern Printing: Right Triangle

```python
rows = 5

for i in range(1, rows + 1):
    print("* " * i)
```

**Output:**
```
*
* *
* * *
* * * *
* * * * *
```

## Accumulator Pattern

A very common technique — use a variable to collect results across iterations:

```python
# Sum of first N natural numbers
n = int(input("Enter N: "))
total = 0

for i in range(1, n + 1):
    total += i

print(f"Sum of 1 to {n} = {total}")
```

## Try It Yourself

1. Print all even numbers between 1 and 50
2. Take a number and calculate its factorial
3. Print a countdown from a user-given number to 1
4. Print a right-angled triangle pattern of any height
5. Find the sum of all digits in a number (e.g., 1234 → 10)

```python
# Factorial Calculator
num = int(input("Enter a number: "))
factorial = 1

for i in range(1, num + 1):
    factorial *= i

print(f"{num}! = {factorial}")

# Sum of Digits
number = int(input("Enter a number: "))
digit_sum = 0
temp = number

while temp > 0:
    digit_sum += temp % 10
    temp //= 10

print(f"Sum of digits of {number} = {digit_sum}")
```

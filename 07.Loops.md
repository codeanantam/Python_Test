# Loops in Python

**(for Loop and while Loop)**

## 1. Introduction

Sometimes we need to repeat a task many times.

Examples:

- Print numbers from 1 to 10
- Print a message 5 times
- Go through each item one by one

Instead of writing the same line again and again, we use **loops**.

Python has two main loops:

- `for` loop
- `while` loop

## 2. The `for` Loop

Used when you know how many times to repeat.

### Using `range()`

`range()` generates a sequence of numbers.

**Print 1 to 5:**

```python
for i in range(1, 6):
    print(i)
```

**Output:**

```
1
2
3
4
5
```

- Start value is included
- End value is excluded

### Different Uses of `range()`

**Only stop value (starts from 0):**

```python
for i in range(4):
    print(i)   # 0, 1, 2, 3
```

**With step:**

```python
for i in range(2, 11, 2):
    print(i)   # 2, 4, 6, 8, 10
```

### Looping Through a String

```python
word = "Code"

for ch in word:
    print(ch)
```

**Output:**

```
C
o
d
e
```

## 3. The `while` Loop

Runs as long as the condition is `True`.

```python
i = 1

while i <= 5:
    print(i)
    i += 1
```

**Output:**

```
1
2
3
4
5
```

**Important:** Always update the variable inside the loop!

### Infinite Loop (Wrong)

```python
i = 1

while i <= 5:
    print(i)
    # Forgot i += 1 → runs forever!
```

Press `Ctrl + C` to stop if this happens.

## 4. Loop Control

### `break` – Stop the loop

```python
for i in range(1, 10):
    if i == 5:
        break
    print(i)
```

**Output:**

```
1
2
3
4
```

### `continue` – Skip and go to next

```python
for i in range(1, 6):
    if i == 3:
        continue
    print(i)
```

**Output:**

```
1
2
4
5
```

## 5. Nested Loops

A loop inside another loop.

```python
for i in range(1, 4):
    for j in range(1, 4):
        print(i, j)
```

This prints all combinations of `i` and `j`.

## 6. Difference Between `for` and `while`

- Use `for` when the number of repetitions is known
- Use `while` when repetition depends on a condition

## 7. Common Mistakes

- Forgetting to update variable in `while` loop
- Using wrong values in `range()`
- Wrong indentation
- Creating accidental infinite loops
